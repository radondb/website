Radondb MySQL Kubernetes released version 2.1.3 on March 24. This version is mainly based on function optimization and upgrade in 2.1.2.

## **Thank**

First of all, thank @andyli029 @acekingke @runkecheng @mgw2168 @molliezhang for submitting the modification.

# **Features List**
1. One click publishing workflow
2. Support rebuilding cluster nodes according to labels
3. Add pod debugging mode
